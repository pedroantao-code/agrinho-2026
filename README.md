# 🌾 Agro Sustentável

Projeto focado no desenvolvimento de uma página web informativa com foco em dicas de preservação ambiental, agricultura consciente e boas práticas no campo. Desenvolvido de forma exclusiva seguindo os critérios de avaliação e rubricas do Programa Agrinho.

## ✨ Funcionalidades
- **Design Responsivo:** Totalmente adaptado para visualização em computadores, tablets e smartphones.
- **Gerador Dinâmico de Dicas:** Um botão interativo rodando via JavaScript que exibe de forma randômica dicas essenciais de manejo sustentável.
- **Formulário com Feedback Realtime:** Validação e captura de dados de contato simulada, respondendo ao usuário na própria interface sem recarregar a tela.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estrutura semântica moderna.
- **CSS3:** Estilização limpa, paleta de cores natural e layouts flexíveis (Flexbox e Grid).
- **JavaScript (Vanilla):** Lógica de interações dinâmicas do usuário.

## 🏷️ Identificação e Tags
- **Contexto:** Projeto Escolar / Competição Educacional.
- **Categoria/Tag:** `agrinho` (identificada nos metadados do `index.html` e elementos visuais do rodapé).
